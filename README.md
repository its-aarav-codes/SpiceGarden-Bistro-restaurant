# SpiceGarden Bistro – Responsive Restaurant Landing Page

Link: https://its-aarav-codes.github.io/SpiceGarden-Bistro-restaurant/

## Overview
SpiceGarden Bistro is a **fully responsive, single-page restaurant website** designed to work smoothly across mobile, tablet, and desktop screens.  
The focus is on **clear layout, fast loading, and real-world usability**, without relying on heavy animations or external frameworks.

This project represents a **production-ready front-end layout** suitable for restaurants, cafés, or small food businesses.

---

## Features
- Mobile-first, responsive design
- Clean hero section with background image handling
- Structured sections for:
  - Introduction / About
  - Menu highlights
  - Why choose us
  - Contact details
- Simple navigation with smooth scrolling
- Lightweight and easy to maintain

---

## Tech Stack
- **HTML5** – semantic and accessible structure  
- **CSS3** – Flexbox-based responsive layout  
- **JavaScript (minimal)** – basic interactivity only  

No backend, no UI frameworks, no unnecessary dependencies.

---

## Design Philosophy
This project prioritizes:
- Readability over visual clutter  
- Consistent layout across screen sizes  
- Practical UX decisions used in real client websites  

The design is intentionally **simple and adaptable**, making it easy to reuse for different restaurant brands.

---

## How to Use
1. Clone or download the repository  
2. Open `index.html` in a browser  
3. Replace text, images, and colors to match your brand  
4. Deploy using any static hosting service (GitHub Pages, Netlify, etc.)

---

## Customization Notes
- Colors and fonts are defined in CSS for easy updates
- Background images can be changed directly in the CSS
- Layout scales cleanly from mobile to large screens
