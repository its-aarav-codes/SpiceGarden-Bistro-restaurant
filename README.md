# SpiceGarden-Bistro-restaurant

A fully responsive, single-page landing website for a modern restaurant concept. Built with **HTML, CSS (Flexbox + Grid), and minimal JavaScript** for mobile navigation and smooth scrolling.

## Overview
This project demonstrates a clean, mobile-first design for a restaurant, showcasing:

- Hero section with call-to-action buttons
- Menu highlights and services section
- Why choose us section with key features
- Testimonials for social proof
- Contact section with WhatsApp and email buttons
- Mobile-friendly navigation with hamburger menu
- Fully responsive layout for all devices

## Features
- **Responsive design**: Mobile-first approach, with layouts adapting seamlessly for tablet and desktop.
- **Clean UI/UX**: Modern typography, clear visual hierarchy, and color palette optimized for readability and user engagement.
- **Interactive elements**: Smooth scroll navigation and mobile menu toggle.
- **Deployment-ready**: Can be hosted on Netlify, GitHub Pages, or any static hosting platform.

## Tech Stack
- HTML5
- CSS3 (Flexbox + CSS Grid)
- JavaScript (minimal, for interactions)

## Color Palette & Typography
- **Primary**: #20462A (deep warm green)  
- **Secondary**: #F5E9DA (soft beige)  
- **Accent**: #F28C28 (rich orange)  

- **Headings**: Playfair Display / Georgia  
- **Body**: Inter / Roboto / system fonts
